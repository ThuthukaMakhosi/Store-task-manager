# Task Manager

## Description
This Python program helps a small business manage tasks assigned to team members. It allows users to add tasks, view assigned tasks, and track completion status through a simple console-based interface.

## Features
- Add tasks for specific users
- View all tasks or tasks assigned to a specific user
- Track task completion status
- Console-based user interaction
- Data stored using text files

## Requirements
- Python 3.x
- Any code editor (VS Code, PyCharm, IDLE, etc.)
